# KH ENG Website

A 20-day English–Khmer lesson platform for foreign teachers in Cambodian international schools.

## Features

- **20 Days of Lessons**: Classroom-focused vocabulary, dialogues, activities, and quizzes
- **Progress Tracking**: Save progress per teacher account
- **Dashboard**: Admin view for tracking all teachers
- **Search**: Find vocabulary across all lessons
- **Audio**: Listen to pronunciation with Web Speech API
- **Dark Mode**: Toggle for comfortable reading

## Deploy to Render

1. Push to GitHub:
```bash
git add .
git commit -m "Ready for production"
git push origin main
```

2. Go to [render.com](https://render.com) and create a new Web Service
3. Connect your GitHub repository
4. Render auto-detects `render.yaml` configuration
5. Click Deploy - your app will be live at `https://kh-eng-website.onrender.com`

The first teacher to register with `whatthap@gmail.com` becomes admin automatically.
